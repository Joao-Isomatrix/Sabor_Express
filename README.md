# Sabor_Express
Projeto de aprendizado sobre Orientação a objetos, classes, e funções

### Projeto
O projeto se trata de uma aplicação responsavel por criar e adicionar restaurantes a uma base de dados (não implementada) e realiza consulta simples, assim como verificar e alterar o estado  de funcionamento do estabelecimento como ativo/inativo.

## Resumo
A aplicação roda centrada em uma classe principal "Restaurante" que é responsavel por criar objetos com as propiedades nome, categoria, avaliação e ativo. no fim da criação do restaurante os dados do objeto são armazenados em uma lista para serem consultados posteriormente.
